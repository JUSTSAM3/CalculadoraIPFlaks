Calculadora web de ips v4

*Esta aplicacion red calcula  la informacion detalla de una direccion IPv4 y su mascara red
Fue desarollada en HTML y Python usando un framework llamado Flask

*Como caracteristicas 
- Calcula las IP de red y broadcast
- Muestra la cantidad de Ips utiles y su rango
- Clasifica la IP
- Detecta si la IP es publica o privada 

*Para su instalacion
Como requisitos se nececitan Python y el Framework Flask
Esto son instalados al ejecutar el archivo "dependencias.sh"

Nota :En este caso dependencias .sh debe tener permosis de sudo


Luego de que las dependencias sean intaladas se debe ejecutar el archivo calculadora.py
